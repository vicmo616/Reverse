The Code for REVERSE

                        
                            //get string from the page
                            //controller function
                            function getValue(){
                                
                                document.getElementById("alert").classList.add("invisible");
                               
                                let userString = document.getElementById("userString").value;
                                
                                let revString = reverseString(userString);
                                
                                displayString(revString);
                            }
                            
                            //reverse the string
                            //logic function
                            function reverseString(userString){
                                let revString = [];
                            
                                //reverse a string using a for loop
                                for (let index = userString.length -1 ; index >= 0; index--) {
                                    revString += userString[index];
                                    
                                }
                                return revString;
                            }
                            
                            //display reversed string to the page
                            //view function
                            function displayString(revString){
                                //write message to the page
                                document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
                                //show the alert box
                                document.getElementById("alert").classList.remove("invisible");
                            }

The Code Is Structured In Three Functions

getValue

getValue accepts(gets) the user input from the page. It utilizes getElementById to pull the values from the page. It then passes the value(s) to the reverseString function. The function reverseString returns an array of values and passes that array to the displayString function. Utilizing again getElementById grabbing the Id of "alert" and adding the invisible class to its classList, we will be undoing this later in the displayString function to show the alert message that this is associated with which is now currently hidden as a result of adding the invisible class.

reverseString

reverseString takes in one parameter being userString, a variable established in the getValue function where we collect the user input of a string. We also create a variable (revString) that holds an array. A for loop is used to generate the user's inputted string and reverse it back to the page by use of decrementing it's length from the last index while each decrement of the userString is being fed to the empty array of revString as the loop makes its way to the start of the string's index. The revString array is then passed into the displayString function.

displayString

displayString takes in one parameter (revString). The variable revString is an array that holds the value of the newly instituted string of userString that was implemented in the ealier mentioned for loop. To write our message to the page we will use getElementById grabbing the "msg" Id with the .innerHTML property to return that HTML content. We then assign that content with the message we wish to present to the user by use of a string literal that holds the value of the revString array. Finally we unhide our alert message we hid earlier in the getValue function, in order to do so we will getElementById once more but instead of adding to the classList, we will remove from it the "invisible" Id that was added earlier to expose our generated message to the user.
